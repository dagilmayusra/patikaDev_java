## Hesaplayan Program
Java ile yarý çapýný kullanýcýdan aldýðýnýz dairenin alanýný ve çevresini hesaplayan programý yazýn.

Alan Formülü : pi * r * r;

Çevre Formülü : 2 * pi * r;

Ödev
Yarýçapý r, merkez açýsýnýn ölçüsü a olan daire diliminin alaný bulan programý yazýnýz.

pi sayýsýný = 3.14 alýnýz.

Formül : (pi * (r*r) * a) / 360
